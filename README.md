## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session


## Getting Started

- Install all dependencies (using the `npm install` command).

- Run the development web server using the `npm start` command.

- Explore the website registering and loggin in.
